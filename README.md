# ⚖️ RightNow – Offline Legal Assistance Platform

## 🚀 Overview  
**RightNow** is a lightweight, offline-first legal assistance web application designed to provide **instant legal guidance** during urgent situations. Built as a **single HTML file**, it ensures accessibility even in **low or no internet connectivity environments**, making it highly reliable in real-world emergencies.

This platform empowers users with **basic legal awareness, guidance, and documentation support** when they need it the most.

---

## 💡 Key Features  

### ⚡ Instant Legal Help  
- Provides quick legal guidance for common situations  
- Helps users understand their rights in real time  

### 🌐 Offline Functionality  
- Works completely **offline**  
- No server, no installation, no dependencies  

### 🧠 Smart RightNow Chat  
- Interactive chatbot interface for legal queries  
- Provides structured responses for user concerns  
- Designed for clarity, simplicity, and speed  

### 📄 Session Recording & Marking *(Planned/Advanced)*  
- Records user interaction sessions  
- Allows marking important moments for later reference  

### 🪶 Lightweight Design  
- Single HTML file  
- Fast loading and minimal resource usage  

---

## 🎯 Problem It Solves  

In many urgent situations, people:  
- Don’t know their legal rights  
- Don’t have immediate internet access  
- Cannot reach a lawyer quickly  

**RightNow bridges this gap** by providing **instant, offline legal assistance**.
